# Linux for Cloud Native

> Advanced Linux, Cloud, DevOps, Containers, and Kubernetes study repository based on the Linux for Cloud Native training from LINUXtips.

## Overview

This repository contains structured documentation, technical notes, hands-on labs, architecture diagrams, and practical exercises developed throughout the Linux for Cloud Native learning path.

The training goes beyond basic Linux administration and focuses on the skills required to operate modern cloud-native platforms, including infrastructure automation, containerized workloads, cloud environments, and production-grade operational practices.
